# Step-by-step Guide


## Hardware Setup
### Required Components:
* ESP8266 NodeMCU
* Arduino Uno
* L298N Motor Driver
* WS2812B RGB LEDS
* 12V DC Motor
* 3.7V 400mA Lithium Battery
* 12V Battery (Power the motor)
* 330<span>&#8486;</span> Resistor
* 9cm &#xD7; 10cm PCB Board
* 4mm Rigid Flange
* Screws and Nuts
